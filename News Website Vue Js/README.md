## Project setup
```
npm install
```

### For installing Bootstrap
```
npm install bootstrap jquery popper.js --save
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
